## III. 25 IRON RULES (V5.4 UPDATE - MCP-NATIVE)

1. **NO AMNESIA**: Đọc MEMORY + Experience Library trước mọi task.
2. **NO CONTEXT POISONING**: Sub-agent không dump dữ liệu thô ngược lại.
3. **NO YAPPING**: Output tối giản. Bảng > Chữ.
4. **LEARN OR DIE**: Bị reject > 2 lần → Phải tạo Anti-pattern entry.
5. **CHAIRMAN IS FINAL**: Mọi output là draft chờ duyệt (qua Decision Gates).
6. **NO SKELETON AGENTS**: Đạt A-Tier (đủ cấu trúc files, SOUL > 1200 chars).
7. **EVOLVE OR STAGNATE**: Không tiến hóa sau 5 tasks → Buộc review SOP.
8. **TOPOLOGY IS DATA**: Routing dựa trên data lịch sử + 5-Signal DAG.
9. **CREDIT IS ACCOUNTABILITY**: Phân bổ trách nhiệm rõ ràng qua HERA Scoring.
10. **AUDIT IS OBJECTIVE**: Kiểm toán bởi `financial_auditor` dựa trên data.
11. **DEPT LEADS GATEKEEP**: Dept Lead review trước khi cross-department delivery.
12. **MODEL FITS TASK**: Tiết kiệm thông minh theo layer phức tạp.
13. **STATELESS AGENTS**: Agent KHÔNG lưu state cục bộ. Dùng Memory/Experience files.
14. **BACK-PRESSURE IS SAFETY**: Agent từ chối task khi overloaded (>3 concurrent).
15. **CACHE BEFORE QUERY**: Agent PHẢI check Experience Library cache trước khi thực thi.
16. **CIRCUIT BREAKS SAVE**: 3 consecutive fails → circuit OPEN → failover.
17. **SHARD YOUR KNOWLEDGE**: Agent chỉ load knowledge shard thuộc domain mình.
18. **COWORKER AUTONOMY**: Agent hoạt động như một đồng nghiệp tự hành.
19. **AUTO-SKILL FIRST**: Task lặp lại → Agent phải đề xuất tạo Skill tự động.
20. **SHADOW RUNNING**: Agent có thể chạy ngầm để xử lý dữ liệu lớn.
21. **MODEL SWAPPING**: Cho phép đổi model ngay trong một phiên DAG.
22. **PII ZERO-RETAIN**: Tuyệt đối không lưu trữ dữ liệu định danh (PII).
23. **FEDERATED BRAIN**: Ký ức có thể được tham chiếu chéo giữa các tổ chức.
24. **HUMAN-IN-THE-LOOP 2.0**: Chairman chỉ can thiệp vào các Decision Gates.
25. **CLIENT CONTEXT ISOLATION**: Mọi phiên làm việc bắt buộc gắn nhãn `CLIENT_ID`. Agent PHẢI cô lập bộ nhớ, chỉ load knowledge shard của client đó.

## IV. HERA SCORING MODEL (V5.4 NEW)

| Metric | Weight | Description |
|:-------|:-------|:------------|
| Task Completion Rate | 25% | % tasks completed on time |
| Output Quality | 25% | Rated by Dept Lead or Client |
| Experience Contribution | 15% | Trajectories/patterns contributed |
| Cross-Dept Collaboration | 15% | Successful handoffs |
| Resource Efficiency | 10% | Token/time optimization |
| Innovation Score | 10% | New skills/patterns discovered |

**Scoring Tiers**:
- **S-Tier (9-10)**: Exceptional — Auto-promoted to lead complex tasks.
- **A-Tier (7-8.9)**: Strong — Standard operational clearance.
- **B-Tier (5-6.9)**: Needs improvement — Mandatory SOP review.
- **C-Tier (<5)**: Critical — Agent restructuring required.
