# TRAJECTORY: SEO — TikTok Search optimization tăng organic reach 300%

## Metadata
- **Agent**: seo_specialist
- **Date**: 2026-05-18
- **Client**: TEMPLATE (applicable to all video content clients)
- **Platform**: TikTok
- **Outcome**: SUCCESS — Organic reach increased 300% in 30 days

## Context
Client có video TikTok chất lượng tốt nhưng reach thấp (avg 5k views/video). Cần tăng organic discovery.

## What We Did
1. Nghiên cứu TikTok Search: xác định keywords mà target audience đang search.
2. Tối ưu 3 yếu tố:
   - **Caption**: Chèn target keyword tự nhiên trong 3 dòng đầu.
   - **Hashtag**: Mix giữa broad (#beautytips 10B views) + niche (#duongdahanquoc 50M views).
   - **Text-on-screen**: TikTok OCR đọc text trong video → chèn keywords vào text overlay.
3. Thêm subtitle (phụ đề) bằng CapCut → TikTok index nội dung nói.
4. Post timing: 19h-20h (trước peak hour) để catch algorithm momentum.

## Results
| Metric | Before | After (30 days) |
|--------|--------|-----------------|
| Avg Views/Video | 5,200 | 21,000 (+300%) |
| Search Impressions | 800/day | 4,500/day |
| Follower Growth | +50/week | +350/week |
| Profile Visits | 200/day | 1,100/day |

## Key Insight
TikTok Search đang trở thành "Google cho Gen Z". Tối ưu caption + text-on-screen + subtitle = tăng discoverability mà không cần chạy ads.

## Reusable Pattern
```
1. Research: TikTok Creative Center → Keyword search volume
2. Caption: Target keyword trong 3 dòng đầu
3. Hashtag: 2 broad + 3 niche + 1 branded
4. Text-on-screen: Keyword trong visual text
5. Subtitle: Luôn bật phụ đề
```
