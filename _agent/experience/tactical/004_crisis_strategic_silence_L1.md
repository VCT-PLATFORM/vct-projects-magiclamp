# TRAJECTORY: Crisis — "Im lặng chiến lược" hiệu quả hơn phản hồi ngay cho L1

## Metadata
- **Agent**: crisis_response
- **Date**: 2026-05-18
- **Client**: TEMPLATE
- **Outcome**: LEARNED — L1 crises resolve faster with strategic silence

## Context
Một client bị 1 review tiêu cực trên Facebook (phàn nàn dịch vụ chậm). Team muốn phản hồi ngay lập tức.

## What We Did
A/B approach trên 2 case tương tự:
- **Case A**: Phản hồi công khai ngay lập tức → Comment thread dài 47 replies, kéo dài 3 ngày, reach tăng 5x (Streisand Effect).
- **Case B**: DM riêng khách hàng xin lỗi + đề nghị bồi thường → Khách xóa review, resolved trong 4 giờ.

## Key Insight
Đối với L1 (phàn nàn đơn lẻ): **Im lặng công khai + xử lý riêng** hiệu quả hơn phản hồi công khai. Phản hồi công khai vô tình tạo sân khấu cho drama.

**Chỉ phản hồi công khai khi**: L2/L3, thông tin sai lệch lan rộng, hoặc ảnh hưởng đến nhiều khách hàng.

## Reusable Pattern
```
L1 (đơn lẻ): DM riêng → xin lỗi → bồi thường → theo dõi
L2 (lan rộng): DM + Official Statement ngắn gọn
L3 (scandal): Full crisis protocol → Chairman involvement
```
